Au cours de cette formation nous allons avoir 2 matiéres

L'intégration

Le HTML

Le HTML sert a créer le contenu du site: il va contenir uniquement la partie textuelle du site

Le CSS

Le CSS va nous permettre d'ajouter la couche graphique a notre site: les couleurs, les formes, le fond, etc.

Le développement

Le Javascript

Javascript est un language de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètres

Le PHP

Javascript est un language de programmation libre, utilisé pour générer des pages web dynamiques qui sont donc modifiés en fonction de certaines informations (heure, nom d'utilisateur, formulaire remplis par l'utilisateur,etc)

MySQL

MySQL est un système de gestion de bases de données, il permet d'enregistrer des informations remplies par l'utilisateur(exemple:formulaire de création de compte) pour ensuite etre ré-utilisé au moment approprié(ex:connexion à un site internet avec identifiant et mot de passe).
