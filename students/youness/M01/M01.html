Au cours de cette formation nous allons avoire 2 matières :

L'integration

Le HTML :

Le HTML sert a crée le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs,les formes, le fond, etc.

Le développement

Le JavaScript :

Javascript est un language de programmation de script employé pour rendre une page web intercative, le comportement de la page est son apparence pourra varier en fonction de certains paramètres.

Le PHP :

Le PHP est language de programmation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines inforamtions (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc).

MySQL :

MySQL est un systeme de gestion de bases de donnees, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de creation de compte) pour ensuite etre ré-utilise au moment approprié (ex: connexion a un site internet avec identifiant et mot de passe).
