L'integration , Le developement , La repartition , jouons avec le CSS !

Au cours de cette formation nous allons avoir 2 matières :

L'integration

Le HTML:

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes,le fond,etc

Le développement

Le javascript:

Javascript est un langage de programmation de script employé pour rendre une page web interactive,le comportement de la page et son apparence pourra varier en fonction de certains paramètres.

PHP

Le PHP est un langage de programmation libre, utilisé pour géneré des pages web dynamiques, qui sont donc modifiés en fonction de certaine information (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc.)

MySQL est un système de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de creation de compte) pour ensuite etre ré-utiliser au moment approprié (ex: connexion à unsite internet avec identifiant et mot de passe).
